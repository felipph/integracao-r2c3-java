# Exemplos de integração com R2C3 - Java

O objetivo desse repositório é ilustrar com exemplos práticos como integrar com o sistema R2C3.

## Canais

### Integração - api rest

Possui exemplos de integração por API REST utilizando bibliotecas.

### Integração - arquivos

Possui exemplos de integração por arquivos.

# Requisitos

* Java 11
* Maven
